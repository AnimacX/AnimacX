## 应用介绍

*目前仅支持`MacOS 14.0`和`iOS 17.0`以上的系统*

### 功能特性

- [x] 弹幕功能：支持手动加载本地弹幕、自动关联云端弹幕
- [x] 视频功能：支持本地视频、网络视频
- [x] 番剧识别
- [x] 发送弹幕
- [x] 番剧资讯和追番
- [ ] 番剧评论评分同步bangumi
- [ ] Pikpak离线观看/下载
