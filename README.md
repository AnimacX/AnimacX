# AnimacX - Anime for mac

>（弹弹Play MacOS版）一款可以追番的弹幕视频播放器

🚀 在Release中浏览最新发布的版本以及更新信息或[直接下载最新版本](https://animacx.pressed.top/api/v1/release/download/latest?s=github.com)

## 应用介绍

*目前仅支持`MacOS 14.0`和`iOS 17.0`以上的系统*

### 功能特性

- [x] 弹幕功能
- [x] 视频功能
- [x] 番剧识别
- [x] 发送弹幕
- [x] 番剧资讯和追番
- [ ] 番剧评论评分同步bangumi
- [ ] 网络视频连接播放
- [ ] Pikpak离线观看/下载
